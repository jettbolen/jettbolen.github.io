Stories of the Land

For two terms of the second year, my group was tasked with developing a larger project for an industry client. Our project was 'Stories of the Land', an ASP.NET Core web application for visitors to Hannin Creek Education Centre, allowing learners to scan a QR code from their mobile device to learn more about the associated plants, along with their "traditional Indigenous uses or culturally significant applications."

Our group corresponded with our client over the course of the year, implementing test-driven development and using an Agile-adjacent system to develop our project, which we presented in full at the end of the year to the client and other industry professionals. I personally implemented the QR code generation functionality, as well as working with various features of ASP.NET Core, Entity Framework Core, Razor pages, and SQLite.

Skills Applied

  • C#, JavaScript, HTML, Bootstrap
  • ASP.NET Core with Entity Framework for handling the base of our application.
  • Razor pages to separate views
  • Cypress for UI and functional testing
  • NUnit for unit testing
  • Deploying the application to Azure for cloud hosting
  • Working in a small team for a long period of time
  • Communicating with a client and gathering information
  • Creating documentation such as class and sequence diagrams, acceptance test procedures, and user scenarios
  • Implementing Test-Driven Development
  • Working with database fixtures for testing

Saskpolytech Hack 2024

For Saskatchewan Polytechnic's first ever 'Hackathon', my team was asked to come up with an idea that would benefit future Saskpolytech students. We used our combined knowledge to create a high-fidelity mockup for a centralized calendar web app that could help students by having everything all in one place. The idea also came with a social RSVP system that would allow students to see what their classmates and friends would be participating in.

This project was made in 8 hours and most of my time was spent working on the main calendar page, creating filterable tags which could be added to events on the calendar.

Skills Applied

  • HTML, CSS, and JavaScript
  • Bootstrap
  • Version Control with Github
  • Working with a small team in a short amount of time
